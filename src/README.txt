📇 Projeto Agenda - Prática de POO

Este projeto foi desenvolvido como uma atividade acadêmica com o objetivo de consolidar os pilares da Programação Orientada a Objetos (POO) em Java. A aplicação simula um sistema de gerenciamento de contatos operado via console.
🎯 Objetivos do Projeto

A atividade foca na aplicação prática de conceitos fundamentais, tais como:

    Abstração e Encapsulamento: Criação da classe Contato com atributos privados e métodos de acesso.

    Instanciação de Objetos: Criação dinâmica de contatos durante a execução.

    Manipulação de Coleções: Uso de ArrayList para armazenamento dos objetos.

    Controle de Fluxo: Implementação de menus interativos com while, seleção de opções com switch-case e tomadas de decisão com if-else.

    Iteração Avançada: Uso do Enhanced For (For-Each) para exibição de dados e Expressões Lambda para remoção lógica de itens.

🛠️ Tecnologias e Ferramentas

    Linguagem: Java

    Entrada de Dados: Scanner (com tratamento de buffer)

    Versionamento: Git

🔄 Estado Atual e Próximos Passos

Atualmente, o projeto funciona em tempo de execução (os dados são armazenados na memória RAM e perdidos ao fechar o programa).

Roadmap de Evolução:
